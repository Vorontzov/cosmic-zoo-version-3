# 🌌 Cosmic Zoo - Finding Belly

A heartfelt platformer adventure game where you search for your beloved pet Belly!

## 🎮 Story

Your dino-frog friend **Belly** has been kidnapped by a mysterious person! Team up with new pet companions as you journey from the sky through the mountains to the dungeon to rescue your best friend.

## 🐾 Characters

| Character | Description | Color |
|-----------|-------------|-------|
| **Belly** | Your beloved dino-frog pet (kidnapped!) | Green |
| **Zoink** | A friendly deer-pig with antlers | Yellow + Pink snout |
| **Zip** | Rude villain who steals your money | Black + White eyes |
| **Linky** | A simple ball creature | Blue |
| **Hang** | A droopy-faced pet | Red |

## 🎯 Level 1 Features

- **Cutscenes** - Full story with dialogue
- **Platforming** - Jump across cloud platforms in the sky
- **Moving Platforms** - Dynamic platforms that move!
- **Zoink's Antler Swing** - Use your companion's ability to swing across gaps
- **Boss Battle vs Zip** - Dodge delayed attacks and the flying pounce (watch for the red circle!)

## 🕹️ Controls

| Key | Action |
|-----|--------|
| ← → or A/D | Move |
| ↑ or W or Space | Jump |
| 3 | Antler Swing (near swing points) |
| 1 | Pounce Attack (in battle) |
| 2 | Gun Attack (in battle) |

## 🚀 Deploy to Vercel

1. Create a [Vercel account](https://vercel.com)
2. Install Vercel CLI: `npm i -g vercel`
3. Run `vercel` in this folder
4. Done! Your game is live!

Or simply drag this folder into Vercel's dashboard!
